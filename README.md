# CC-Plugin-linux-desktop

Claude Code plugin for linux-desktop slash commands and agents.

## Installation

```bash
/plugin marketplace add danielrosehill/CC-Plugin-linux-desktop
/plugin install linux-desktop@danielrosehill
```
